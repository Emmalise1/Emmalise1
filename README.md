I'm a passionate software engineer building impactful technology for African communities. I specialize in creating accessible, inclusive applications that solve real-world problems.

###  Current Project: NexKin
A daily health habit platform for Rwandan youth that:
- Delivers health challenges via web and USSD
- Rewards engagement with mobile airtime
- Supports users with disabilities
- Reaches users on any phone, with or without internet
